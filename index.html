git使用教程

git本地操作

  • git init 初始化
  • git status 查看更改状态
  • git add 添加至缓存区
  • git commit -m "说明"
  • git reflog 查看版本信息
  • git reset -- hard HEAD <版本号> 退回操作
  • git checkout -- 退回，只修改但未add
  • git reset HEAD   退回，已经add，但未add
  • git rm 删除操作

github

  • git remote add origin
  • git push -u origin master 第一次推送master分支上所有内容时
  • git push origin master 推送最新修改
  • git clone 克隆远程仓库

git branch

  • git checkout -b 创建分支
  • git branch 查看所有分支
